# Block View Roadmap

This is the **product-level repository** for BlockNote. It focuses on product goals & user stories rather than implementation details.

## Purpose

This repository is used to:
- Track product goals and outcomes
- Capture user stories and product intent
- Plan product direction and priorities

## Issue Templates

- **Product Goal**: For defining high-level product objectives
- **User Story**: For describing user problems and intended outcomes

> ⚠️ **Note**: Creating product goals and user stories are for core maintainers only.

## Reporting Bugs or Enhancements

For bugs or enhancement requests in the BlockNote core code, please use the [BlockNote core repository](https://github.com/TypeCellOS/BlockNote/issues).
